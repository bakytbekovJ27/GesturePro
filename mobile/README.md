# GesturePro Mobile

Мобильное приложение-компаньон для GesturePro, разработанное на **React Native / Expo**.

Приложение выступает в роли беспроводного пульта управления презентацией, позволяя загружать файлы с телефона и перелистывать слайды.

## Стек технологий

- **Фреймворк**: Expo SDK 51, Expo Router
- **Интерфейс**: React Native Paper, Lucide React Native
- **Управление состоянием**: Zustand
- **Сетевые запросы**: Axios

## Основные возможности

- **Сопряжение (Pairing)**: Подключение к активной десктопной сессии по 6-значному PIN-коду.
- **Загрузка файлов**: Загрузка документов презентаций (`.pdf`, `.pptx`) напрямую из памяти мобильного устройства на бэкенд-сервер.
- **Управление показом**: Перелистывание слайдов вперед/назад, отслеживание текущего номера слайда и имени файла.

## Быстрый старт

### Установка зависимостей

1. Перейдите в каталог мобильного приложения:
   ```bash
   cd mobile
   ```

2. Установите npm-пакеты:
   ```bash
   npm install
   ```

### Запуск приложения в режиме разработки

1. Запустите локальный сервер Expo:
   ```bash
   npx expo start
   ```

2. На телефоне установите приложение **Expo Go** (доступно в Google Play и App Store).
3. Сканируйте QR-код из терминала с помощью камеры телефона (iOS) или через приложение Expo Go (Android).

## Настройка подключения к бэкенду

Для корректной работы сопряжения мобильное устройство и компьютер с бэкендом должны находиться в одной локальной Wi-Fi сети. 
Укажите LAN IP-адрес вашего компьютера в файле `.env` мобильного приложения в переменной:
```env
EXPO_PUBLIC_API_URL=http://<ВАШ-LAN-IP>:8000/api/v1
```
(Пример шаблона содержится в [mobile/.env.example](file:///Users/aidin07/Documents/GesturePro/mobile/.env.example)).
