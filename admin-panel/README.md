# GesturePro Admin Panel

Панель управления (Admin Panel) для проекта GesturePro, разработанная на **React + TypeScript + Vite**.

Этот веб-интерфейс позволяет администраторам и персоналу отслеживать текущие презентации, управлять пользователями, активными сессиями ( pairing PINs) и просматривать общую статистику системы.

## Основные возможности

- **Панель статистики (Dashboard)**: Общие показатели зарегистрированных пользователей, активных сессий сопряжения, количество загруженных презентаций и график распределения по статусам.
- **Управление пользователями (User Management)**: Список пользователей с поиском, возможностью создания новых аккаунтов персонала (staff), смены паролей, деактивации и удаления пользователей.
- **Мониторинг сессий (Session Monitoring)**: Список всех созданных сессий сопряжения, отображение активных PIN-кодов и возможность удаленно деактивировать/закрывать сессии.
- **Управление презентациями (Presentation Management)**: Отслеживание загруженных файлов (`.pdf` / `.pptx`), статусов их конвертации (`pending`, `processing`, `ready`, `failed`) и удаление некорректных презентаций.

## Стек технологий

- **Фреймворк**: React 19, TypeScript
- **Сборщик**: Vite 8
- **Маршрутизация**: React Router v7
- **Стилизация**: CSS-переменные, современный темный стеклянный интерфейс (Glassmorphism)
- **Запросы к API**: Axios с автоматическим обновлением JWT-токенов при истечении
- **Графики**: Recharts

## Быстрый старт

### Требования
- Установленный Node.js 20+
- Запущенный Django-backend (по умолчанию ожидается на `http://127.0.0.1:8000`)

### Установка зависимостей и запуск

1. Перейдите в каталог панели администратора:
   ```bash
   cd admin-panel
   ```

2. Установите npm-пакеты:
   ```bash
   npm install
   ```

3. Запустите dev-сервер:
   ```bash
   npm run dev
   ```

Приложение запустится на `http://localhost:5174`.

### Сборка для продакшена

Для сборки оптимизированного дистрибутива выполните:
```bash
npm run build
```
Файлы сборки будут помещены в каталог `dist/`.

## Настройка проксирования и API

В режиме разработки Vite настроен на автоматическое проксирование запросов `/api` и `/media` на адрес локального бэкенда Django (`http://127.0.0.1:8000`). Конфигурация прокси описана в файле [vite.config.ts](file:///Users/aidin07/Documents/GesturePro/admin-panel/vite.config.ts).

## Авторизация

Панель администратора использует авторизацию на базе JWT-токенов (`SimpleJWT`). Для входа в интерфейс пользователь должен обладать статусом персонала (`is_staff=True` в Django).
Если у вас нет учетной записи, создайте суперпользователя в бэкенде:
```bash
python manage.py createsuperuser
```
И введите данные этого аккаунта на странице входа панели.
