# Gesture Suite

Полноценное приложение для управления презентациями с помощью жестов. Поддерживает PDF-файлы, раскадровки из папок и рисование поверх слайдов с корректным отображением русского языка.

## Возможности
- **Мультиформатность**: Загрузка PDF-презентаций или наборов изображений из папки `slides/`.
- **Жестовое управление**: Полноценное бесконтактное управление как в меню, так и во время презентации.
- **Аннотации**: Рисование поверх слайдов, ластик и быстрая очистка (все сохраняется для каждого слайда отдельно).
- **Нативная интеграция**: Использование нативных диалоговых окон для выбора файлов в macOS/Windows/Linux.
- **Локализация**: Поддержка русского и английского языков с корректными шрифтами.

## Структура проекта

```
gesture_suite/
│
├── main.py                        ← Точка входа в приложение
│
├── core/
│   ├── gesture.py                 ← Детектор жестов (MediaPipe + логика Пинча)
│   ├── localization.py            ← Система перевода (RU/EN)
│   ├── renderer.py                ← Отрисовка текста (Pillow для UTF-8) и HUD
│   └── slide_loader.py            ← Логика загрузки PDF (PyMuPDF) и папок
│
├── screens/
│   ├── menu_screen.py             ← Главное меню
│   ├── settings_screen.py         ← Выбор языка
│   ├── load_screen.py             ← Экран выбора источника слайдов
│   └── presentation_screen.py    ← Основной интерактивный экран презентации
│
└── slides/                        ← Папка для разделенных слайдов (PNG/JPG)
```

## Установка и запуск

1. **Клонируйте репозиторий** (или скачайте файлы).
2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Запустите приложение**:
   ```bash
   python main.py
   ```

## Управление

### 1. Интерфейс (Меню / Нагрузка)
- **Указательный палец**: Движение курсора.
- **Пинч (смыкание большого и указательного)**: Нажатие кнопки (удержать 0.7с).
- **Клавиши**: 
    - `1` — Старт
    - `2` — Настройки
    - `B` — Назад
    - `O` — Выбор PDF через проводник
    - `D` — Загрузка папки `slides/` или Демо.

### 2. Презентация
- **✊ Кулак (1 сек)**: Активация / Пауза системы.
- **☝ Указательный палец**: Рисование.
- **👍 Большой палец**: Ластик.
- **🖐 Раскрытая ладонь (1.5 сек)**: Полная очистка рисунка на текущем слайде.
- **✌ "Виктория" (2 пальца) + Смах**: Переход между слайдами (влево/вправо).

**Клавиши во время презентации:**
- `←` / `→` (A / D): Листать слайды.
- `O`: Выбрать другой файл (переход к экрану загрузки).
- `C`: Сменить цвет кисти.
- `X`: Очистить текущий слайд.
- `S`: Сохранить скриншот с вашими пометками.
- `+` / `-`: Изменить размер кисти.
- `M`: Вернуться в главное меню.
- `Q` / `ESC`: Выход из программы.
